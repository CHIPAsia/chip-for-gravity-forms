# CHIP for Gravity Forms

This module adds the CHIP payment method option to your Gravity Forms.

[![WordPress Plugin Check](https://github.com/CHIPAsia/chip-for-gravity-forms/actions/workflows/plugin-check.yml/badge.svg?branch=main)](https://github.com/CHIPAsia/chip-for-gravity-forms/actions/workflows/plugin-check.yml)

## Installation

* [Download the zip file of the plugin.](https://github.com/CHIPAsia/chip-for-gravity-forms/archive/refs/heads/main.zip)
* Log in to your WordPress admin panel and go to **Plugins** â†’ **Add New**
* Select **Upload Plugin**, choose the zip file you downloaded in step 1, and press **Install Now**
* Activate the plugin

## Configuration

### Global Configuration

Set the **Brand ID** and **Secret Key** in the plugin's settings.

### Form-specific Configuration

You can override the Global Configuration by setting it in the form-specific configuration.

## Refund

Refunds made through the WordPress Dashboard can only be made in full amount. The success of the refund execution means the refund has been successfully triggered to the CHIP API. The completion of the refund depends on the CHIP refund policy.
